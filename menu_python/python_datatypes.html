Numbers



  • There are three numeric types in Python:
    • Integer
    • floating point
    • complex
  • Integers can be of any length.
  • Floating point numbers are separated by decimal points.
    • 1 is an integer, 1.0 is a floating point number.

Example

    >>> x = 1
    >>> print(x)
    1
    >>> y = 0.1
    >>> print(y)
    0.1


Strings



  • Strings are defined either with single quotes or double quotes.
    • ‘hello’ is the same as “hello”.
  • The difference between the two is that using double quotes makes it easy to include apostrophes. “it’s python”
  • Python does not have a character data type: a single character is simply a string with length 1.
  • Strings can be output to screen using print().

Example

    >>> print('hello')
    hello
    >>> str1 = "hello world"
    >>> print(str1)
    hello world
    >>> str2 = "It's Python"
    >>> print(str2)
    It's Python
