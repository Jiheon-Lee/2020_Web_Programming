Python Character



platform independent

Regardless of OS such as Windows, Linux, Android, MacOS, and iOS, you can use the program once you write the program.


Interpreter Language

The source code is interpreted at the time of execution without any separate translation process so that the computer can handle it. It is simple to write and requires less memory but is slow to execute.


object oriented

The program is written based on the unit module (object) rather than the order of execution.


dynamic typing language

Determine the type of data the program should use at the time the program runs.
