Python Install



To use Python, you need to install it first.
You can install the version you want on Python's official website.
However, we recommend using Python3.x or later.
The link to Python's official website is as follows.


https://www.python.org/downloads/



Once the installation is complete, you can see the following screens in the Start menu: Click IDLE to run.




If you look at the following screen, it's a success! A lab environment has been created for Python!
