Variable



  • A variable is a named memory space for storing data.
    • Each variable must have a unique name called an identifier.
    • A variable is a container that holds data which can be changed later in the program.

  • In Python, a variable is a reference (pointer) to an object.
    • A variable does not directly store the object.
    • A variable stores the pointer to the object.

  • In Python, variables need not to be declared or defined.
    • To create a variable, you just assign it a value and then start using it.
    • Assignment is done with a single equals sign (=).

Example

    >>> name = 'john'
    >>> x = 10
    >>> y = 20
