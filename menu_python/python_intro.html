What is Python?



Python is an advanced programming language published in 1991 by programmer Guido van Rossum.


  • platform independent
  • Interpreter Language
  • object oriented
  • dynamic typing language
  • At first, it was made with C language

The Python name originated from "Monty Python's Flying Circus," and Python's logo overlaps two snakes.
