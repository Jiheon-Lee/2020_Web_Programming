if-else

if-else

                if (boolean_expression)
                statement_1;
                else
                statement_2;

-explain-

If (boolean_expression) is TRUE (1),
execute statement_1;
Otherwise FALSE(0),
execute statement_2.

example 

                if ( grade >= 60 ) {
                    printf( "Passed\n" );
                    }
                else {
                    printf( "Failed\n" );
                    }
  1. grade >=60 is True -> print "Passed"
  2. gread >=60 is False -> print "Failed"
