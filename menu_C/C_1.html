-explain-

  1. #inlcude <stdio.h>
    • include a file calld 'stdio.h'
    • Think that In/Output functions can be used by using this.
  2. int main()
    • function main begins program execution
    • Think that almost all functions go in here.
  3. {}
    • { - start of function
    • } - end of function
    • Braces must have a beginning and an end together.
  4. printf("");
    • Instructs computer to perform an action - prints the string of characters within quotes (" ")
    • All statements must end with a semicolon (;)
  5. \n
    • It is newline character
  6. return 0;
    • 'return 0' is 'To inform the computer that the program has been successfully terminated.'
    • If not 0 but 1, the computer know that the program did not end successfully.
