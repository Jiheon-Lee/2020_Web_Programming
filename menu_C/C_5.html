Loop - for, while

for

                for (loop_var = n; loop continuation condition; loop_var = loop_var + increment)
                loop_statement;
                “loop_statement” may be a compound statement

while

                while (boolean_expression)
                {
                loop_statement;
                boolean_expression-altering statement;
                }
                “loop_statement” may be a compound statement

example 

                int sum = 0;

                for(int i = 0; i <= 10; i++) {
                    sum += i;
                }
                printf("%d",sum);
  1. (0<=10) is True, { } is executed. i= 0+1 = 1
  2. (1<=10) is True, { } is executed. i= 1+1 = 2
    3. .
    .
    .
  3. (10<=10) is False, and the program goes to the next statement. 
                int sum = 0;
                int i  = 1;

                while(i <= 10) {
                    sum += i;
                    i++;
                }
  1. (0<=10) is True, { } is executed. i= 0+1 = 1
  2. (1<=10) is True, { } is executed. i= 1+1 = 2
    3. .
    .
    .
  3. (10<=10) is False, and the program goes to the next statement.
