-explain-

  1. /*~~*/
    • To be completely ignored by compilers and used only for human convenience.
    • All characters between /**/ are ignored.
  2. //
    • The use of this is similar to the one before(/*~~*/).
    • // For annotations in form, only lines with // are annotated.
