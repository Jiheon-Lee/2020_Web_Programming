CSS-Text



Text Color

The color property is used to set the color of the text. The color is specified by:
  • a color name - like "red"
  • a HEX value - like "#ff0000"
  • an RGB value - like "rgb(255,0,0)"
Look at CSS Color Values for a complete list of possible color values.
The default text color for a page is defined in the body selector.


Why Use CSS?

CSS is used to define styles for your web pages, including the design, layout and variations in display for different devices and screen sizes.

Example

<head>
<style>
body { color: blue; } h1 { color: green; } </style>
</head>
<body>

<h1>This is a heading.</h1>
<p>This is a paragraph.</p>

</body>
</html>

NOTE:For W3C compliant CSS: If you define the color property, you must also define the background-color.



Text Color and Background Color


In this example, we define both the background-color property and the color property:

For W3C compliant CSS: If you define the color property, you must also define the background-color.

Example

<head>
<style>
body { background-color: lightgrey; color: blue; } h1 { background-color: black; color: green; } </style>
</head>
<body>

<h1>This is a heading.</h1>
<p>This is a paragraph.</p>

</body>
</html>
